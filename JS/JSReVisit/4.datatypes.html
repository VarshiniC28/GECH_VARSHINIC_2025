1. Number

A primitive data type used to represent both integer and floating-point numeric values in JavaScript.

Example: let age = 25; let price = 19.99;

2. String

A primitive data type that represents a sequence of characters enclosed in quotes, used for text data.

Example: let name = "John"; let message = 'Hello World';

3. Boolean

A primitive data type that can only have two values: true or false, commonly used in conditional statements.

Example: let isActive = true; let isEmpty = false;

4. Undefined

A primitive data type that represents a variable declared but not assigned any value.

Example: let x; console.log(x); // undefined

5. Null

A primitive data type that represents the intentional absence of any object value.

Example: let user = null;

6. BigInt

A primitive data type that allows representation of integers larger than the maximum safe integer limit (2^53 - 1).

Example: let bigNumber = 12345678901234567890n;

7. Object

A non-primitive data type that stores collections of key-value pairs and serves as the foundation for complex data structures.

Example: let person = {name: "John", age: 30};

8. Symbol

A primitive data type that creates a unique and immutable identifier, used as object keys to avoid naming conflicts and enable privacy in object properties.

Example: let sym = Symbol("id"); let obj = {[sym]: 123};
